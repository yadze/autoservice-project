# autoservice-project

Full Stack веб-приложение для учёта заказов на шиномонтажные работы (смена, ремонт, сезонное хранение шин).  

## Команда
- **Backend-разработчик**: Вадим 
- **Frontend-разработчик**: Артём

## Технологический стек
- **Backend**: Python, Flask (основной сервис), FastAPI (вспомогательный сервис для аналитики)
- **Frontend**: HTML, CSS, JavaScript 
- **База данных**: PostgreSQL 
- **Аутентификация**: JWT 

## Функциональные требования (выполнено/в работе)
- [ ] Регистрация и вход пользователей с JWT
- [ ] Защищённые роуты
- [ ] Три связанные сущности в БД (пользователи, автомобили, заказы) – в процессе
- [ ] Поиск, фильтрация, сортировка, пагинация (SQL-уровень)
- [ ] CRUD для заказов и автомобилей
- [ ] Два микросервиса (Core + Supporting) с разными фреймворками
- [ ] Эндпоинт `/dashboard` с диаграммами (Chart.js)
- [ ] Эндпоинт `/about` (HTML) и `/api/about` (JSON)
- [ ] Эндпоинт `/api/hash/{str}`
- [ ] Репликация Master-Slave
- [ ] Резервные ответы при недоступности сервисов
